# FTP-SD-STM32G4

## Environment
### Software:
- STM32CubeIDE 1.16.1
- FatFS R0.12c
- IoLibrary 1.0.1

### Hardware:
- STM Nucleo64 G474
- Ethernet W5500 Module
- Adafruit SD CARD Shield

Additional, this implementation uses functions from the following PR: https://github.com/Wiznet/ioLibrary_Driver/pull/148
